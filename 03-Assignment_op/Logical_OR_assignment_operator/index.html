Logica OR assignment operator

In JavaScript, the logical OR (||) operator returns the first truthy value found in an expression. If both values are falsy, it returns the last value.
The OR assignment operator (|=) is a short-hand for assignment that assigns the result of the logical OR operation to a variable.
The code snippet below demonstrates the usage of logical OR assignment operator. 


Syntax ->

x ||= y
or 
x = x || y
*/


Code ->

let name = {
    first_name : "Ritesh",
    last_name : "",
};

console.log(name.first_name);   

name.first_name ||= "Divyanka";

console.log(name.first_name);  //this can't change first name because it has truthy value 

console.log(name.last_name);

name.last_name ||= "Pagare";

console.log(name.last_name);  //this changes last name to pagare because it has falsy value
